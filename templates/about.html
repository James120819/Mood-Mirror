Mood Mirror is an emotional tracking app that allows users to check their mood, write reflections, and also review emotional patterns over time.

Features:

  • Daily mood tracking
  • Optional reflection journaling
  • Mood history list
  • Visual mood summary chart
  • Exportable mood data

Tech Stack:

  • Python (Flask)
  • HTML / CSS / JavaScript
  • Chart.js for data visualization
  • SQLite / SQL database for mood history

This project was created by James Farrow as a personal mental health software project that focuses on wellbeing, UX, and data visualization.
